# Memory map

Here are some brief details about what is where, in the original build.

## Basic layout

| Address | Label |
| -- | -- |
| $403c | Working data |
| $4082 | TKP, keyboard handler. First code |
| $423f | Entry point |
| $4322 | Final byte of code |
| $4332 | Display data (DFILE) |
